Different Functions

Step 1. Kindly open console.
log()- Outputs a log to the console
Alert()- Outputs an Alert box to the page
error()- Outputs an error box to the console
time()- Starts a timer
timeEnd()- Ends the timer
warn()- Outputs a warning message to the console
clear()- Clears the console
assert()- Writes an error message to the console if the assertion is false
