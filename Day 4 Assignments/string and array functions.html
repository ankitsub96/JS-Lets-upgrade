String and Array Functions

substring()- This method extracts the characters in a string between "start" and "end", not including "end" itself.

splice()- Adds/removes items to/from an array, and returns the removed item(s).
Syntax is
`array.splice(index, howmany, item1, ....., itemX)`, where index specifies where elements are inserted and howmany specifies how many elements of original array are removed.
replace()- Outputs an error box to the console
Syntax is
string.replace(searchvalue, newvalue)
